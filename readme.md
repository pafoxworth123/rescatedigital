# 📼 RescateDigital.com.mx

**Preserva tus recuerdos para las futuras generaciones.**

Sitio web profesional para el servicio de digitalización de recuerdos antiguos (VHS, Super 8, fotografías, negativos, cassettes) con enfoque en el mercado mexicano.

## 🌐 Sitio en Vivo

[https://rescatedigital.com.mx](https://rescatedigital.com.mx)

---

## 📋 Descripción del Proyecto

RescateDigital es un sitio web de conversión diseñado para ayudar a las familias mexicanas a preservar sus recuerdos en formatos antiguos (VHS, Super 8, fotos impresas, negativos, cassettes) digitalizándolos antes de que se deterioren.

El sitio está inspirado en el modelo de negocio de Legacybox.com pero adaptado para el mercado mexicano con:
- Precios en pesos mexicanos (MXN)
- Contenido en español mexicano
- Sin envíos prepagados (el cliente paga el envío)
- Ubicación física en Las Ánimas, Puebla

---

## 🚀 Tecnologías Utilizadas

- **HTML5** - Estructura semántica y SEO
- **CSS3** - Diseño responsive, animaciones, efectos visuales
- **JavaScript Vanilla** - Interactividad (menú móvil, FAQ, formulario)
- **Font Awesome** - Íconos profesionales
- **Cloudflare Pages / GitHub Pages** - Hosting recomendado
